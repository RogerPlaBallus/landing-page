# Landing Page

A minimal, dark-theme React landing page built with Vite.

Production target: Vercel root hosting.

Legacy GitHub Pages build: `npm run build:gh-pages`

Live GitHub Pages URL:
https://rogerplaballus.github.io/landing-page/

## Features

- Responsive layout with Navbar, About, Skills, Projects, Contact
- Smooth scroll-to-top and cursor-trail effects
- Tailwind-styled components (no external UI libraries)

## Getting Started

```bash
# Install dependencies
npm install
# Run locally
npm run dev
# Build for Vercel
npm run build
# Build for GitHub Pages
npm run build:gh-pages
```
